 mem.hProject wide memory managerment. Uses MEM_* This provides the function definitions for manipulating heaps and heap managers which allow for all of the dynamic memory allocation to be done at once. The heap data structure also provides memory recycling is needed. The goal of the memory management is to prevent the need to allocate memory more than once during the life time of a program. Instead all of the memory is allocated in predefined heaps which serve as arrays of objects. Memory can also be saved and loaded from disk
